# Discord Bot Installer CLI

CLI para instalar bots de Discord de OpceanAI automáticamente.

## Instalación
```bash
npm install -g @guita/discord-bot-installer
```

## Uso
```bash
discord-bot list
discord-bot install nebula
discord-bot install archan
```

## Bots Disponibles
- **Nebula**: Bot multipropósito con música
- **Archan**: Bot de IA con Gemini
